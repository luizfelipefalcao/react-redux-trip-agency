

This project was bootstrapped with Create React App.


## What is inside?

This project uses lot of stuff as:

- ReactJs
- React-Redux
- React-Saga
- React-Router-Dom
- CSS
- React-Icons
- Immer
- Json-Server


## Getting Started

```bash
npm install
# or
yarn install
to install the application

and 

npm run start
# or
yarn start
to run the application

yarn json-server server.json -p 3333
to run json-server

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
Open [http://localhost:3333](http://localhost:3333) with your browser to see the api result.

## Commands

- `start`: starts the application on `localhost:3000`
- `json-server`: starts the api application on `localhost:3333`
